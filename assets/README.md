# Martenweave Site Assets

The canonical brand source is `martenweave-mark.svg`. The public logo, favicon, app
icons, and social preview images are derived from that mark without changing the
logo design.

`github-social-preview.png` is intentionally reserved for the GitHub repository
social preview setting. It is not referenced by page metadata because Open Graph
and Twitter use channel-specific preview images.

The social images use the existing Martenweave mark, brand colors, and the approved
open-source model registry positioning.

`marketing/` contains two original editorial PNGs created with OpenAI image generation
for this project: `model-atlas-hero.png` and `readiness-signal.png`. They contain no
product UI, customer data, or claims. The public [Marketing Assets](/docs/assets.html)
page provides approved use guidance and download links.
