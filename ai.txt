Martenweave is a backend-first agentic data model registry for SAP migration, master data management, data governance, data quality, and AMS/support teams.

Martenweave is a backend-first model registry that converts scattered model knowledge into validated, traceable, AI-ready model truth for migration, MDM, data governance, and support teams.

Current core package: martenweave-core 0.4.1.

Current capabilities include canonical Markdown/YAML model files, deterministic validation, generated SQLite/JSONL indexes, search/query, trace/impact analysis, dataset profiling, gap detection, health and governance reports, ownership/audit reports, CSV/XLSX review flows, PatchProposal to ChangeRequest workflows, issue drafts, GitHub-ready change bundles, local API, and MCP-ready backend behavior.
