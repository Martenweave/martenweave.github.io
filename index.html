Available now

A real core, not a landing-page promise.

`martenweave-core 0.4.1` is a Python package and CLI with canonical files, deterministic validation, generated indexes, gap detection, impact analysis, examples, and reviewable AI proposals.

Run quickstart See examples 
python -m pip install martenweave-core
martenweave validate --repo examples/customer_bp_model
martenweave build-index --repo examples/customer_bp_model --jsonl
martenweave impact FEP-S4-KNVV-KDGRP --repo examples/customer_bp_model
