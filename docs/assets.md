# Marketing Assets

This public asset library gives partners, contributors, presenters, and writers a small set of accurate Martenweave visuals. Use the official project name, preserve the artwork, and link to [Martenweave](https://martenweave.github.io/) or the [core repository](https://github.com/metalhatscats/martenweave-core) when practical.

## Use guidance

- Do not imply SAP affiliation, certification, customer endorsement, or a capability not documented by the project.
- Do not alter the logo, remove marks from visual assets, or use the images to present third-party products as Martenweave.
- The two editorial PNGs were created with OpenAI image generation for this project; use them as whole images, with optional surrounding copy added outside the image.
- The architecture SVG and brand marks are source assets in this repository. Keep their labels and meaning intact.

For an asset request, a presentation-specific crop, or a collaboration use case, start with [Partnerships](/docs/partnerships.html).
