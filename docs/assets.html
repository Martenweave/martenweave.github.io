Marketing Assets

This public asset library gives partners, contributors, presenters, and writers a small set of accurate Martenweave visuals. Use the official project name, preserve the artwork, and link to Martenweave or the core repository when practical.

Model atlas

An editorial background for explaining canonical model truth, lineage, and controlled change. It contains no product UI, customer data, or textual claims.

Readiness signal

Use this visual for readiness, evidence, validation, review, and partnership conversations. It intentionally leaves clear space for a presentation headline added by the user.

Architecture scheme

This exact diagram explains the product boundary: inputs become canonical model files; deterministic validation and generated indexes create inspectable intelligence; AI proposals remain reviewable; humans approve changes.

Use guidance

  • Do not imply SAP affiliation, certification, customer endorsement, or a capability not documented by the project.
  • Do not alter the logo, remove marks from visual assets, or use the images to present third-party products as Martenweave.
  • The two editorial PNGs were created with OpenAI image generation for this project; use them as whole images, with optional surrounding copy added outside the image.
  • The architecture SVG and brand marks are source assets in this repository. Keep their labels and meaning intact.

For an asset request, a presentation-specific crop, or a collaboration use case, start with Partnerships.
